13. 练习：While 循环

练习：最接近的平方数

写一个 while 循环，用于计算比整数 limit 小的最大平方数，并将其存储在变量 nearest_square 中。平方数是整数乘以自己后的积，例如 36 是一个平方数，因为它等于 6*6。

例如，如果 limit 是 40，你的代码应该将 nearest_square 设为 36。

Start Quiz:

nearestsquare.py 
limit = 40

# write your while loop here


print(nearest_square)
