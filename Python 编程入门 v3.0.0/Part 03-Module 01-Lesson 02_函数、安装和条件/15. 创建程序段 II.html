15. 创建程序段 II

练习解决方案：创建帮助函数 convert_to_numeric

这可以是一个非常短的函数：我们只是将分数转换为浮点数。 

def convert_to_numeric(score):
    """
    Convert the score to a float.
    """
    converted_score = float(score)
    return converted_score

编写此函数最简单的一个方法是： 

def convert_to_numeric(score):
    """
    Convert the score to a float.
    """
    return float(score)

两种方式都完全可行！

总而言之，在 scores_to_rating() 中跳过这个函数也是合理的，只需将其替换为 

    score1 = float(score1)
    score2 = float(score2)
        …


因为运算非常简单，代码行内即可完成。
但是，此时我们将问题分解并且运用帮助函数，而不是在 scores_to_rating() 中的解决，若将来输入中再加入一些其他复杂内容，我们也可以轻松搞定。
