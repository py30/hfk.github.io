配置自己的 Python 编程设置

大家可以在屏幕上看到我的设置，请你们花一点时间熟悉自己的电脑。如果想尝试某个文本编辑器，可以试试以下这些我们最喜欢的文本编辑器。

对于 Windows：

  • Notepad++

对于 Linux：

  • Atom
  • Sublime Text
  • emacs

将文本编辑器、终端 / 命令行和网络浏览器中的 Udacity 教室设置在自己的屏幕上，以便于不断迭代 Python 脚本。使用显示设置来找到你认为最舒服的显示方式，并查看是否可以找到一个“将 tab 转换为 4 个空格”的选项，该设置对 Python 缩进非常有用。

编辑 Python 脚本

在文本编辑器中打开 my-first-script.py 并查看。

编辑 welcome_message() 函数定义中的标准消息。保存编辑内容并在终端中运行： 

$ python3 my-first-script.py

即可看到你的修改内容。

接下来，尝试编辑传递给 welcome_message() 函数的参数。你想欢迎谁？

通过在脚本中添加更多打印语句和更多函数，充分体验你在新编码环境中学到的内容！在脚本中填写注释，方便以后回顾时，可以查看自己执行了哪些操作。

从现在开始，我们建议大家以此种方式尝试所有代码。将代码放入文本编辑器中的新文件（不要忘记添加一个 .py 扩展名），确保自己位于终端 / 命令行中的正确目录，然后开始运行！你可能会想为每个新的练习或想法创建一个新文件。

编写自己的脚本

在文本编辑器中打开一个新的空文件，命名并保存在 Python 学习的文件位置。然后将以下代码编入其中。 

how_many_snakes = 1
snake_string = """
Welcome to Python3!

             ____
            / . .\\
            \  ---<
             \  /
   __________/ /
-=:___________/

<3, Philip and Charlie
"""


print(snake_string * how_many_snakes)
