17. 创建程序段 IV

练习解决方案：构建帮助函数 score_to_rating_string

我们按照以下方法编写此函数： 

def score_to_rating_string(av_score):
    """
    Convert the average score, which should be between 0 and 5, 
    into a rating.
    """
    if av_score < 0:
        rating = None
    elif av_score < 1:
        rating = "Terrible"
    elif av_score < 2:
        rating = "Bad"
    elif av_score < 3:
        rating = "OK"
    elif av_score < 4:
        rating = "Good"
    elif av_score <= 5:
        rating = "Excellent"
    else:          #Using else at the end, every possible case gets caught
        rating = None
    if rating == None:
        return "Error: not in correct range."
    else:
        return rating

还有另外一个解决方案是，在实际运行中使用多个返回语句：针对每一个不同情况返回不同语句。在该解决方案中，使用 else 意味着这个函数始终可返回值。 

def score_to_rating_string(av_score): 
    """
    Convert the average score, which should be between 0 and 5, 
    into a rating.
    """
    if  av_score < 0:
        return "Error"
    elif av_score < 1:
        return "Terrible"
    elif av_score < 2:
        return "Bad"
    elif av_score < 3:
        return "OK"
    elif av_score < 4:
        return "Good"
    elif av_score <=5:
        return "Excellent"
    else:
        return "Error"

练习：组合 scores_to_rating 函数

先前所有帮助函数都构建得不错！现在可以将所有这些函数组合到一起，使其成为一个整体进行运行。

在该编程练习中，将编写的每个帮助函数放入编码窗口，然后在编写函数 scores_to_rating 时使用。请注意，此函数以五个分数（可能不是数字）开始，取中间三个分数的平均值，并将该平均值转换为返回的书面评分。

需要确保将一个函数的输出正确传递到 scores_to_rating 函数体的下一个输入。一次添加一个，同时还需添加有用的注释，而且调用 print 进进行测试，将有助于控制错误！
