12. 复合数据结构

字典词典

我们来回顾一下 elements 字典， 

elements = {'hydrogen': 1, 'helium': 2, 'carbon': 6}

该字典将元素名称（字符串）映射到其原子序号（整数）。但是如果想要存储关于每个元素的更多信息，如它们的重量和符号，会出现什么结果呢？可以通过调整字典，来将元素名称（字符串）映射到存储该数据集合的字典中： 

elements = {'hydrogen': {'number': 1, 'weight': 1.00794, 'symbol': 'H'},
            'helium': {'number': 2, 'weight': 4.002602, 'symbol': 'He'}}

可以采用与之前相同的方法，使用方括号或 get 方法查看该嵌套字典中关于条目的信息： 

>>> print(elements['helium'])
{'number': 2, 'symbol': 'He', 'weight': 4.002602}
>>> print(elements.get('unobtainium', 'There\'s no such element!'))
There's no such element!

可以采用如下所示的方法从氦字典查找具体的信息： 

>>> print(elements['helium']['weight'])
4.002602

这段代码首先在元素字典中查找关键字 "helium"，生成氦字典。第二个查询是 ['weight'] ，然后再查找该氦字典中的关键字 "weight” 即可查找氦的原子量。

练习：在嵌套字典中添加值

尝试使用嵌套字典。在 elements 字典的每个字典中添加另一个条目 'is_noble_gas'。插入新条目后，即可执行查找： 

>>> print(elements['hydrogen']['is_noble_gas'])
False
>>> print(elements['helium']['is_noble_gas'])
True
