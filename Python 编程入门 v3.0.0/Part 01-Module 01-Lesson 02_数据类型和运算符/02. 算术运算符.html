02. 算术运算符

算术运算符

算术运算符

算术运算

  • + 加
  • - 减
  • * 乘
  • / 除
  • % 取模（相除后的余数）
  • ** 取幂（注意 ^ 并不执行该运算，你可能在其他语言中见过这种情形）
  • // 相除后向下取整到最接近的整数

关于按位运算符的更多信息请参阅 此处 。Python 也遵守一般的数学运算顺序，你可以访问以下网页以复习这方面的知识： https://en.wikipedia.org/wiki/Operation_(mathematics)。

关于取模运算，更多信息请参考 这里 。值得注意的是，取模运算和取余运算是一个非常容易混淆的概念。
