20. 练习：列表方法

练习： len 、 max 、 min 和列表

此页面底部有一个 python 环境，你可以利用该环境运行与此页面上的任何练习相关的测试代码！

len、max 和 min

以下代码的输出是什么？（请将这道多选题的答案中的逗号当做换行符。） 

a = [1, 5, 8]
b = [2, 6, 9, 10]
c = [100, 200]

print(max([len(a), len(b), len(c)]))
print(min([len(a), len(b), len(c)]))
SOLUTION: 4, 2

练习： sorted 、 join 和列表

sorted 和 join

以下代码的输出是什么？（请将这道多选题的答案中的逗号当做换行符。） 

names = ["Carol", "Albert", "Ben", "Donna"]
print(" & ".join(sorted(names)))
SOLUTION: Albert & Ben & Carol & Donna

练习： append 和列表

以下代码的输出是什么？（请将这道多选题的答案中的逗号当做换行符。） 

names = ["Carol", "Albert", "Ben", "Donna"]
names.append("Eugenia")
print(sorted(names))
SOLUTION: ['Albert', 'Ben', 'Carol', 'Donna', 'Eugenia']
