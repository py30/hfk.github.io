13. 类型和类型转换

Type & Type Conversion

类型和类型转换

你到目前为止，已经见过四种数据类型：

  1. 整型
  2. 浮点型
  3. 布尔型
  4. 字符串

你在之前的视频中，简单了解了 type() ，它可以用来检查你所处理的任何变量的数据类型。 

>>> print(type(4))
int
>>> print(type(3.7))
float
>>> print(type('this'))
str
>>> print(type(True))
bool

你发现，你可以更改变量类型以执行各种不同的操作。例如 

"0" + "5"

结果完全与以下代码的不一样 

0 + 5

你认为以下代码的结果是什么？ 

"0" + 5

下面的代码呢： 

0 + "5"

检查变量类型非常重要，可以确保在编程时你所获的结果是你想要的结果。
