练习：计算

在这道练习中，你需要通过编写 Python 程序来帮助一位铺瓦工做一些计算。
地面上有两个区域需要贴砖，其中一个区域的大小为 宽 9 块砖，长 7 块砖 ，另一个区域的大小为 宽 5 块砖，长 7 块砖 。此外，6块砖头组成一包。
请编写程序计算以下问题：

  1. 这位铺瓦工贴完这两个区域一共需要多少块砖？
  2. 如果购买 17 包砖，且每包有 6 块砖 。铺完两个区域之后，还剩下多少块砖？

Start Quiz:

tiles.py 
# Fill this in with an expression that calculates how many tiles are needed.
print()

# Fill this in with an expression that calculates how many tiles will be left over.
print()

SOLUTION:
  • `print((17 - 6)%(5 + 2))`
  • `print(4/2 - 7*7)`
